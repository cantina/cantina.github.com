attach

  • options
)

Defined in node_modules/cantina/lib/plugins/http.js:27

Attach the plugin to an application.

If the app is using amno, then the app's start() method will start up an amino service with a name and version based on package.json or the options.

Otherwise, the app will be started as regular flatiron http app.

Parameters:

  • options Object

    Application options. If the app is NOT using amino, then host and port can be specified here.
